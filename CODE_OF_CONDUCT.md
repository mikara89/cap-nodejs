# Code of Conduct

CAP for NestJS aims to be a welcoming, professional project for users,
contributors, and maintainers.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being clear about problems.
- Keep technical criticism focused on the work, not the person.
- Welcome beginners and explain project context when it helps.
- Respect maintainer time by keeping issues and pull requests focused.

## Unacceptable Behavior

- Harassment, threats, hate speech, or personal attacks.
- Dismissive or hostile language toward people or groups.
- Sharing private information without permission.
- Repeated off-topic or disruptive behavior after being asked to stop.
